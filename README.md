## 🔴 Business Problem  

Blinkit, a quick‑commerce platform, needs to understand **sales performance, outlet efficiency, and customer satisfaction**.  
Without structured Excel analysis:  
- ❌ Revenue drivers remain unclear  
- ❌ Outlet performance can’t be benchmarked  
- ❌ Customer feedback is anecdotal, not data‑driven  

➡️ The goal is to analyze Blinkit’s dataset using **Excel (Pivot Tables, Charts, and KPIs)** to uncover actionable insights.

---

## 🎯 Task (My Role as Analyst)

| Task | Objective |
|------|-----------|
| 🧹 **Data Preparation** | Import and clean Blinkit dataset |
| 📊 **Pivot Tables** | Summarize sales by category, outlet, and location |
| 📈 **Charts & KPIs** | Visualize revenue, discounts, and ratings |
| 🏪 **Outlet Analysis** | Compare performance by type and size |
| 📑 **Dashboard** | Build interactive Excel dashboard |

---

## ⚙️ Action (What I Did)

╔══════════════════════════════════════════════════════════════════════╗
║   🧹 Data Cleaning   |   📊 Pivot Tables   |   📈 Dashboard Design    ║

- 🧹 **Data Cleaning** → Removed duplicates, handled missing values, standardized categories  
- 📊 **Pivot Tables** → Created summaries for revenue, discounts, and ratings  
- 📈 **Charts & KPIs** → Designed bar charts, line charts, and KPI cards  
- 🏪 **Outlet Analysis** → Segmented outlets by size, type, and city tier  
- 📑 **Dashboard** → Built interactive Excel dashboard with slicers  

---

## 📈 Results  

- ✅ **Total Revenue** → ~$1.20M in sales  
- ✅ **Top Categories** → Fruits, Dairy, and Snacks contributed ~60% of revenue  
- ✅ **Discount Effect** → Moderate discounts (10–20%) maximized demand  
- ✅ **Outlet Performance** → Large urban outlets outperformed smaller ones  
- ✅ **Customer Ratings** → Average satisfaction ~3.9 across all products
